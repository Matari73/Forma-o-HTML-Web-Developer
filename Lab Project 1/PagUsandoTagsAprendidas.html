O que é HTML?

(voltar)

HTML(Hypertext Markup Language) é uma linguagem de marcação usada para criar páginas web. Ela é composta por um conjunto de tags ou elementos que são utilizados para definir a estrutura e o conteúdo de uma página, como títulos, parágrafos, imagens, links, entre outros elementos.

Tags

(voltar)

As marcações em uma página são feitas por meio de tags, que geralmente são estruturas de linguagem de marcação que contêm instruções. Essas instruções são interpretadas pelos navegadores, que as utilizam para renderizar a página. As tags são comumente utilizadas como delimitadores de estilo ou de conteúdo, agrupando o conteúdo de acordo com o tipo, forma e formatação. Isso permite que outras linguagens, como CSS, possam ser utilizadas para aplicar estilos ao conteúdo. Exemplos:

  • html: Define o início do documento HTML.
  • head: Define informações sobre o documento, como o título da página, metadados e links para scripts e estilos.
  • p: Define um parágrafo de texto.
  • img: Insere uma imagem na página.

Atributos

(voltar)

Os Atributos são usados para fornecer informações adicionais sobre os elementos de uma página web, comotags e suas respectivas propriedades. Os atributos são definidos dentro das tags e especificam características, como a cor do texto, tamanho da fonte, alinhamento, entre outros aspectos. Os atributos são compostos por um nome e um valor, separados por um sinal de igual (=). O valor pode ser uma string entre aspas duplas ou simples, ou pode ser um valor booleano. Quando um atributo é definido com um valor booleano, a presença do atributo em si é suficiente para ativar a propriedade correspondente.

Por exemplo, a tag "img" pode ser usada para exibir uma imagem na página, e pode incluir atributos como:

  • src: Especifica o caminho para o arquivo de imagem.
  • alt: Define um texto alternativo que é exibido quando a imagem não pode ser carregada.
  • width e height: Define a largura e altura da imagem, respectivamente.

Títulos

(voltar)

Os Títulos são tags utilizadas para definir o título da página ou seção. Existem seis níveis de títulos disponíveis, representados pelas tags h1 até h6, onde o h1 é o título mais importante e o h6 é o menos importante.

Parágrafos

(voltar)

Os Parágrafos são utilizados para separar blocos de texto em unidades lógicas e coesas. Eles são representados pela tag p, que é usada para definir um novo parágrafo. São importantes para tornar o texto da página mais legível e organizado. Eles ajudam a quebrar o texto em pedaços menores e mais fáceis de ler, o que torna a leitura mais agradável para os usuários.
